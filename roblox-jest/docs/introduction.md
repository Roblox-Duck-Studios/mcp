# Roblox Jest

Roblox Jest is a testing framework for Roblox TypeScript (roblox-ts) projects. It is based on Jest v27.4.7 and provides a familiar testing experience for Roblox game development.

## Overview

Roblox Jest brings the power and flexibility of Jest to the Roblox platform. It allows you to write comprehensive unit tests for your Roblox TypeScript code, ensuring your game logic works correctly across all scenarios.

## Key Features

- **Jest v27.4.7 Compatible**: Uses the same API and patterns as the standard Jest framework
- **Roblox Integration**: Designed specifically for roblox-ts and Rojo workflows
- **Multiple Test Environments**: Support for SERVER, CLIENT, and SHARED test configurations
- **Code Coverage**: Built-in coverage reporting with @rbxts/coverage
- **CLI Tool**: jestrbx CLI for running tests against Roblox places

To start using Roblox Jest in your project:

1. Install the required packages
2. Set up your test configuration
3. Create your first test file
4. Run tests with jestrbx
